# 12954. x만큼 간격이 있는 n개의 숫자

문제 주소: [주소](https://programmers.co.kr/learn/courses/30/lessons/12954)

```c++
//코드 입력
#include <string>
#include <vector>

using namespace std;

vector<long long> solution(int x, int n) {
    vector<long long> answer;
    
    int adder = x;
    for(int i=0;i<n;++i){
        answer.push_back(adder);
        adder += x;
    }
    
    return answer;
}

```
