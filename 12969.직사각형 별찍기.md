# 12969.직사각형 별찍기

문제 주소: [주소](https://programmers.co.kr/learn/courses/30/lessons/12969)

```c++
//코드 입력
#include <iostream>

using namespace std;

int main(void) {
    int row,col;
    std::cin>>col>>row;
    for(int i=0;i<row;++i){
        for(int j=0;j<col;++j){
            std::cout<<"*";
        }
        std::cout<<"\n";
    }
}
```
