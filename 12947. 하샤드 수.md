# 12947. 하샤드 수

문제 주소: [주소](https://programmers.co.kr/learn/courses/30/lessons/12947)

```c++
//코드 입력
#include <string>
#include <vector>
#include <iostream>

using namespace std;

bool solution(int x) {
    
    std::string number = std::to_string(x);
    int sum = 0;
    for(int i=0;i<number.length();++i){
        sum += (number[i] -'0');
    }
    
    return x % sum == 0;
}
```



### 풀이방법

std::to_string() 을 이용하여 숫자를 String으로 변환 시켰다.
